Get a device token

For clients that can't pair themselves — like an iOS Shortcut, which has no way to run the pairing exchange on its own.

Paste the pairing token from your dashboard's Devices screen. This page exchanges it for a bearer token and shows it to you once — it isn't saved anywhere by this page, so copy it before you leave.

Device token

Paste this into your Shortcut's Authorization header, as Bearer <token> — see the README for the exact steps. It won't be shown again; if you lose it, come back here and generate a new one (this doesn't invalidate the old one, so revoke it from the dashboard's Devices screen afterward if you don't want both around).
